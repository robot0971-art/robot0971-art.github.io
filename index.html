About Me

안녕하세요

Unity와 C#을 이용해 게임 플레이의 재미를 코드로 구현하는 개발자 이선규입니다.

Unity 기반으로 멀티플레이 낚시 게임, 방치형 게임, 생존 게임 등 다양한 장르를 직접 설계하고 개발했습니다.
네트워크 동기화, UI/UX 흐름, 게임 데이터 구조까지 클라이언트 전반을 다뤘습니다.

최근에는 C#과 LLM API를 결합한 데스크톱 에이전트 도구(AgentQ)를 개발하며,
복잡한 비즈니스 로직을 직관적인 사용자 경험으로 연결하는 능력을 키웠습니다.

아이디어를 빠르게 검증하고, 빠른 프로토타이핑과 체계적인 리팩토링 사이의 균형을 중시합니다.
필요한 기술이 있다면 직접 찾아보고 프로젝트에 바로 적용하며 익히는 것을 좋아합니다.
